RESIDENT EVIL OUTBREAK

SUBJECT: PCSX2 ONLINE CONFIGURATION PROTOCOL

STATUS: CLASSIFIED

PHASE: 07 STEPS

Einsatzbesprechung

Dieses Tutorial erklärt Schritt für Schritt, wie du RESIDENT EVIL Outbreak File 1 & File 2 online mit dem PCSX2 Emulator spielst. Die Online-Einstellungen sind bereits vorbereitet.

Voraussetzungen & Warnhinweise

  • [!] Die Spiele sind japanische Versionen, die auf Deutsch gepatcht wurden.
  • [!] Ein Controller wird dringend empfohlen (PlayStation oder Xbox).
  • [!] Die Online-Einstellungen im Emulator sind bereits vorkonfiguriert.

DATA RETRIEVAL: COLLECTION

01

Dateien entpacken

Die RAR-Datei entpacken. Enthalten sind: PCSX2 Emulator, die Spiele und ein japanisches BIOS.

02

Account auf dem Outbreak-Server erstellen

Auf der Community-Seite einen Account erstellen:

HTTPS://OBSRV.ORG/UCP.PHP?MODE=REGISTER

HINWEIS: Login-Daten unbedingt aufschreiben oder sicher speichern.

03

PCSX2 – Erster Start

  • PCSX2 über pcsx2-qt.exe starten.
  • Beim ersten Start das Update durchführen.
  • Anschließend auf Spieleverzeichnis hinzufügen klicken und den Ordner Games auswählen.
  • Den Scan der Spiele mit Ja bestätigen.
